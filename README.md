# Android CourseNext App

## Description

This Android application is engineered to streamline the process of assignment submission, fostering an efficient and seamless educational experience. It serves as a digital bridge between teachers and students, simplifying the exchange of academic tasks and responses. Teachers can upload announcements, assignments and students can submit completed assignments through the app. The app uses Firebase email authentication along with Google authentication for secure sign-in.

## Features

- **Teacher Interface**: Teachers can create new classes and upload announcements & assignments for each class.
- **Student Interface**: Students can join classes using class codes, read announcements and can submit their completed assignments.


## Technologies Used

- **Kotlin**: The app is written in Kotlin, a statically typed programming language for modern multiplatform applications.
- **Firebase**: Used for backend services like authentication and database management.
- **Google Authentication**: Used for secure sign-in.
- **Android Studio**: The app is developed using Android Studio.
